# Chess Master - チェス

[← メインページへ戻る](README.md)

完全なルールのチェスゲーム。ローカルで友達と対戦するか、AIと対戦できます。

![チェス盤](chess-game-initial.png)

## 機能

- **二人対戦** - 同じデバイスで二人でプレイ
- **AI対戦** - AIと对战（Minimax法 + アルファベータ枝切り、深さ3探索）
- **完全なチェスルール**：
  - キャスリング（长短）
  - アンパッサン（取りこぼし）
  - ポーンプロモーション（Queen、Rook、Bishop、Knightを選択）
- チェック、チェックメイト、ステalemate検出
- 代数的記法による棋譜
-投了機能
- 简体中文ベースUI

## 使い方

1. `index.html` をブラウザで開く
2. または `npx serve .` でサーバーを起動し http://localhost:3000 にアクセス

## 操作ボタン

| ボタン | 機能 |
|--------|------|
| 人機対弈 | AIと対戦 |
| 双人对弈 | 二人で対戦 |
| 新游戏 | 新しいゲーム |
| 悔棋 | 一手戻す |
| 投降 | 投了 |

## AIについて

AIは以下の技術を使用：
- **Minimax法** + アルファベータ枝切り
- **深さ3探索** - パフォーマンスと強さのバランス
- **位置評価表** - 駒の配置を評価
- **駒価値表** - 各駒の基本価値

## 技術スタック

- 純粋なバニジャ JavaScript、依存関係なし
- 単一HTMLファイル（CSS + JS込み）
- Unicodeチェス駒
- CSS Gridレイアウ卜
